# Twitter2
Twitter2 is mini SNS system extended by Memopad project.

## Installation
Type follow commands...  
`$ git clone https://github.com/19K0111/Twitter2.git`  
`$ cd Twitter2`  
`$ rails s`  
Open Web browser and go to http://127.0.0.1:3000/
